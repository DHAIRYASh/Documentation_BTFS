Active Orders

This component displays a list of orders fetched from the "orders" collection in the Firestore database.

DEPENDENCIES

  • React
  • useState
  • useEffect
  • Firebase (Firestore)
  • "./Slidebar" Component (make sure the path is correct)
  • "./Orders.css" Stylesheet

METHODS

formatDate(timestamp)

This method takes a timestamp and returns the formatted date in "MM/DD/YYYY" format.
Parameters: timestamp (Timestamp): The timestamp to format.

Returns: Formatted date (String)

 formatTime(timestamp)

This method takes a timestamp and returns the formatted time in "HH:MM:SS AM/PM" format.
Parameters: timestamp (Timestamp): The timestamp to format.
Returns: Formatted time (String)

formatAddress(address)

This method takes an address object and returns the formatted address string.
Parameters: address (Object): The address object to format.
Returns: Formatted address (String)

fetchOrders()

This method is called inside the useEffect hook to fetch the orders from the Firestore "orders" collection.

useEffect

The useEffect hook is used to fetch the orders from the Firestore database when the component mounts. It calls the fetchOrders method and updates the orders state with the fetched data.

RENDERED OUTPUT

The component renders a table containing the list of orders. Each order is displayed in a table row with the following columns:

  • Serial Number (S/N)
  • Email
  • Name
  • Address
  • Date
  • Time
  • Order ID
  • Book Names
  • Total Amount
