Component: DiscoverBooksPage

Dependencies

React, react-responsive-carousel, react-router-dom

CSS Dependencies

"DiscoverBooksPage.css", "react-responsive-carousel/lib/styles/carousel.min.css"

Imports

import React from "react";
import { Carousel } from "react-responsive-carousel";
import "./DiscoverBooksPage.css";
import { useContext, useState } from "react";
import { CartContext } from "./CartContext";
import booksData from "./booksData";
import { useNavigate } from "react-router-dom";

Description

A component that displays a carousel of book banners, the best seller book, and a carousel of upcoming books. The component also has functionality to add books to the cart.

State

The state of this component includes:

  • showPopup: A boolean to control the visibility of the added-to-cart popup.

Hooks

This component uses the following React hooks:

  • useContext to access the CartContext
  • useState to manage the showPopup state
  • useNavigate to navigate to different routes

Variables

  • banner01 (string): Image URL for the book cover.
  • banner02 (string): Second Image URL for the book cover.
  • book01 (string): Image URL for the first upcoming book.
  • book02 (string) : Image URL for the second upcoming book.
  • book03 (string) : Image URL for the third upcoming book.
  • book04 (string): Image URL for the fourth upcoming book.
  • book05 (string): Image URL for the fifth upcoming book.
  • best (string): Image URL for the best seller book cover.
  • navigate (function) : Navigates to a different route.

    • Functions

    handleAddToCart function to add books to the cart and show a popup confirmation.

    Rendered JSX
