Address Form

This is a React component representing a form for entering a shipping address. It contains input fields for first name, last name, email address, address line 1, address line 2 (optional), city, state/province/region, zip/postal code, and country.

USAGE

import React, { useState } from "react"; import "./AddressForm.css"; export default function AddressForm({ onChange }) { // Component implementation }

PROPS

onChange (function): A callback function that is triggered whenever the input fields in the form are changed. It takes two arguments: name (string) representing the name of the input field that was changed, and value (string) representing the new value of the input field.

STATE

emailValid (boolean): A state variable that represents whether the entered email address is valid or not.

METHODS

  • handleInputChange(e: Event): void: A method that is called when an input field in the form is changed. It updates the state and triggers the onChange callback function.
  • validateEmail(email: string): boolean: A helper method that performs simple email validation using regular expression. It takes an email address as input and returns a boolean value indicating whether the email address is valid or not.

The input fields are rendered within a grid container with each field wrapped in a grid-item div.

INPUT FIELDS

  • firstName: Represents the first name input field.
  • lastName: Represents the last name input field.
  • email: Represents the email address input field. It has an additional invalid class applied if the entered email is not valid.
  • address1: Represents the address line 1 input field.
  • address2: Represents the address line 2 input field (optional).
  • city: Represents the city input field.
  • state: Represents the state/province/region input field.
  • zip: Represents the zip/postal code input field.
  • country: Represents the country input field.
