Book Data

DESCRIPTION

The booksData.js file contains an array of book objects that represent different books. Each book object has the following properties:

  • id: The unique identifier of the book (number).
  • title: The title of the book (string).
  • author: The author of the book (string).
  • description: A brief description of the book (string).
  • price: The price of the book (string).
  • cover: The URL of the book's cover image (string).

USAGE

To use the booksData array, import it into your JavaScript file like this:

import booksData from './booksData';

You can then access the array and its elements using standard JavaScript array operations

EXAMPLE

Here is an example of how you can access the booksData array and its elements in your code:


            import booksData from './booksData';
        
            // Iterate over each book
            booksData.forEach((book) => {
                console.log(`Title: ${book.title}`);
                console.log(`Author: ${book.author}`);
                console.log(`Price: ${book.price}`);
                console.log('---');
            });

This will print the title, author, and price of each book in the booksData array.

NOTES
  • The booksData array can be expanded by adding more book objects to it.
  • The cover property contains the URL of the book's cover image. It uses the process.env.PUBLIC_URL environment variable to provide the base URL for the image.
