Review Form

This component displays the order review, including shipping address, payment details, order items, and total amount.

  • React
  • useContext from React
  • "./Review.css" (CSS file)
  • CartContext from "./CartContext" (Adjust the import path as needed)
  • booksData from "./booksData" (Make sure to adjust the import path as needed)

PROPS

  • cartItems: The items in the cart (array)
  • total: The total amount of the order (number)
  • addressInfo: The shipping address information (object)
  • firstName: The first name of the recipient (string)
  • lastName: The last name of the recipient (string)
  • address1: The first line of the address (string)
  • city: The city of the address (string)
  • zip: The zip code of the address (string)
  • country: The country of the address (string)
  • paymentInfo: The payment details (object)
  • cardName: The name on the card (string)
  • cardNumber: The card number (string)
  • expDate: The expiration date of the card (string)

EXAMPLE


                import React from "react";
                import Review from "./Review";
                
                const cartItems = [
                  {
                    id: 1,
                    title: "Book 1",
                    quantity: 1,
                    price: 10.99
                  },
                  {
                    id: 2,
                    title: "Book 2",
                    quantity: 2,
                    price: 15.99
                  }
                ];
                
                const total = 42.97;
                
                const addressInfo = {
                  firstName: "John",
                  lastName: "Doe",
                  address1: "123 Main St",
                  city: "New York",
                  zip: "10001",
                  country: "USA"
                };
                
                const paymentInfo = {
                  cardName: "John Doe",
                  cardNumber: "4111111111111111",
                  expDate: "12/22"
                };
                
                function App() {
                  return (
                    <div>
                      <h1>Order Review</h1>
                      <Review cartItems={cartItems} total={total} addressInfo={addressInfo} paymentInfo={paymentInfo} />
                    </div>
                  );
                }
                
                export default App;
