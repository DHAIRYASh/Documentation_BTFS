ProfilePage Component

A React component for rendering the profile page of a user.

Imports


                import React, { useState, useEffect } from 'react';

                import { useAuth } from '../config/AuthContext';

                import { useNavigate } from 'react-router-dom';

                import { doc, getDoc } from 'firebase/firestore';

                import db from '../config/firebase-config';

                import './ProfilePage.css';

Component Definition

ProfilePage() is a functional component that renders the profile page.

Variables

currentUser : The current authenticated user.

logout : A function to log out the user.

sendResetPasswordEmail : A function to send a reset password email.

navigate : A function to navigate to a different page.

userDetails : The details of the user.

resetEmailStatus : The status of the reset password email.

ed2 : The URL of the user's profile picture.

ed3 : The URL of the second card image.

ed4 : The URL of the third card image.

displayName : The display name of the user.

firstname : The first name of the user.

lastname : The last name of the user.

logoutImageUrl : The URL of the logout image.

useEffect()

An effect hook that fetches user details from Firestore when the current user changes.

handleResetPassword()

A function that sends a reset password email to the current user's email address. Sets the reset email status based on whether the email was sent successfully or not.

userLogout()

A function that logs out the current user and navigates to the login page.

Rendered JSX

  • A div element with a class of "profilePage-container" that contains the following elements:
  • A div element with a class of "profilePage-welcome-header" that contains an h1 element with the text "Welcome {displayName}".
  • A div element with a class of "profilePage-user-details-section" that contains the following elements:
  • A div element with a class of "profilePage-profile-picture-section" that contains an img element with the source set to the value of ed2 and alt text set to "Profile" and a class of "profilePage-profile-picture".
  • A div element with a class of "profilePage-user-info" that contains the following elements:
  • A p element with the text "First Name: {firstName}"
  • A p element with the text "Last Name: {lastName}".
  • A p element with the text "Email: {currentUser.email}".
  • A div element with a class of "profilePage-card" that contains an img element with the source set to the value of ed4 and alt text set to "Cart" and a class of "profilePage-cardImage" and a button element with the text "Go To Cart" and an onClick event handler that navigates to '/cart'.
  • A div element with a class of "profilePage-card" that contains an img element with the source set to the value of ed3 and alt text set to "Reset Password" and a class of "profilePage-cardImage" and a button element with the text "Reset Password" and an onClick event handler that calls the handleResetPassword function and a p element with the class "profilePage-status-message" and text set to resetEmailStatus if it exists.
  • A div element with a class of "profilePage-card" that contains an img element with the source set to the value of logoutImageUrl and alt text set to "Logout" and a class of "profilePage-cardImage" and a button element with the text "Logout" and an onClick event handler that calls the userLogout function.
