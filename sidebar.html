Sidebar Component

This component is responsible for rendering the sidebar menu in the dashboard.

Import Statements


import React from "react";
import { NavLink, useLocation } from "react-router-dom";
import "./Slidebar.css"; // Import the sidebar styles

Function Component


const Sidebar = () => {
    ...
};
export default Sidebar;

Hooks

  • useLocation: Hook from react-router-dom to access the location object.

Helper Function

  • checkIsActive(path): Determines if the current route matches the provided path.

JSX Markup


return (
    <div className="sidebar">
        <NavLink to="/dashboard" className={({ isActive }) => "nav-link" + (isActive ? " active" : "")} end>
            Dashboard
        </NavLink>
        <NavLink to="/dashboard/orders" className={checkIsActive("/dashboard/orders") ? "nav-link active" : "nav-link"}>
            Orders
        </NavLink>
        <NavLink to="/dashboard/users" className={checkIsActive("/dashboard/users") ? "nav-link active" : "nav-link"}>
            Users
        </NavLink>
    </div>
);

Sidebar Structure

The Sidebar component renders a <div> element with the class name "sidebar". Inside the <div>, there are three <NavLink> components representing different menu items:

  • The first <NavLink> to "/dashboard" uses a function to conditionally add the "active" class.
  • The second and third <NavLink> components use the checkIsActive function to determine the active route.

The Sidebar component is exported as the default export of the module.
